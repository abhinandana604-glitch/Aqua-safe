# AquaSafe - Smart Community Health Surveillance System

AquaSafe is a smart community health surveillance system developed using Python and Flask to manage patient health data, analyze disease patterns, and identify high-risk areas in communities.

The system provides a centralized dashboard for managing patient information, village-level health analytics, and risk monitoring to support better healthcare decision-making.

---

## Features

### User Authentication
- User registration and login system
- Secure user access management

### Patient Management
- Add new patients
- View patient details
- Edit patient information
- Delete patient records
- Search and filter patient data

### Health Analytics Dashboard
- Total patient count
- Village statistics
- Water source analysis
- Disease distribution visualization
- High-risk area identification

### Village Health Monitoring
- Village-wise patient analysis
- Identification of high-risk villages based on patient data

### Data Visualization
- Interactive charts using Chart.js
- Disease and village analytics representation
